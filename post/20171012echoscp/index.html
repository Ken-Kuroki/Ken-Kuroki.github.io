Echoing in .bashrc clashes scp data transfer

Posted on 2017, Oct 12

One day when I tried to run scp to transfer data between my terminal and a server, the command somehow didn’t work. It was caused by echo in .bashrc. You have to make it conditional to interactive mode so that scp runs without the problem.

Reference: http://stackoverflow.com/questions/12440287/scp-doesnt-work-when-echo-in-bashrc

So, instead of just echoing, do like this:

if [[ $- =~ "i" ]]; then
    echo something
fi
